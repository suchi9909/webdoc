Set of Rules

Copyright Rules

  • Don’t use images from any source like google. Use images from free sites only like freepik , pixabay etc. (ref. links are above)
  • Don’t copy code or design from any sources like themeforest or dribble. You can copy code from free sites only like codepen, w3school etc.

HTML Rules

  • Don’t write inline CSS or style
  • Don’t use such tag line <br> tag and <hr> tag etc.
  • All style CSS files will be in the head tag and the script file will be at the end of the page.
  • Custom CSS file and js file should be at the end of all files.
  • Make a habit to write comments in every section or the elements which defines something like form or brand logo.
  • Write code in proper format like header part in <header> tag, all the section will be in the <section> tag, and footer content will be in <footer> tag
  • Always indent your code to make easy structure. You can use vs code extension for that.
  • You can use h1 tag once in the page which will be at the top of the site.

CSS Rules

  • Make proper and relatable classname.
  • Use proper format for classname for example if you use “_” or “-“ , use it in a whole project.
  • Make a habit to write comments.
  • Write index or content at the top of the file to make proper format of styles.
  • Defines all the css property of h1 to h6, ul, p at the starting of the stylesheet.

Bootstrap Rules

  • Don’t override any bootstrap class. Make custom class for making change.
  • Never use CDN for bootstrap files.
